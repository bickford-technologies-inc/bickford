# Bickford Neuropsych Assessment Suite

This package implements the neuropsychological assessment and health monitoring suite for Bickford AI deployments.

## Features

- Automated working memory, RAG recall, canon enforcement, and audit integrity tests
- Scheduled health checks for all deployments
- Generates a "Neuropsych Profile" dashboard and compliance reports
- Integrates with the Bickford ledger and canon enforcement modules

## Usage

- Import and schedule the assessment runner in your deployment pipeline or CI/CD
- View results in the dashboard or export compliance reports

## Files

- `assessment.ts` — Main assessment runner and test definitions
- `dashboard.ts` — Dashboard generation and visualization helpers
- `report.ts` — Compliance report generator
