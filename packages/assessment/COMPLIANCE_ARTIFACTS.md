# Bickford Compliance Artifacts

This document describes the compliance artifacts generated by the Bickford Neuropsych Assessment Suite for regulatory and audit purposes.

---

## 1. Compliance Report

- **Generated by:** `report.ts`
- **Contents:**
  - Date/time of assessment
  - Pass/fail status for each neuropsychological domain (working memory, RAG recall, canon enforcement, audit integrity)
  - Details and timestamps for each test
- **Purpose:**
  - Satisfies SOC-2, ISO 27001, and DoD audit requirements for cryptographic proof of system health and compliance

## 2. Neuropsych Profile Dashboard

- **Generated by:** `dashboard.ts`
- **Contents:**
  - Tabular summary of all assessment results
  - Visual pass/fail indicators
  - Timestamps for each domain
- **Purpose:**
  - Enables ongoing monitoring and preventive maintenance
  - Provides auditors and customers with a clear, visual compliance status

## 3. Tamper-Evident Ledger

- **Generated by:** `@bickford/ledger`
- **Contents:**
  - Cryptographically hashed, chained log of all actions and assessment results
- **Purpose:**
  - Enables external, third-party verification of compliance and system health

---

## How to Use

- Run the assessment suite on a schedule (see `schedule.ts`)
- Export compliance reports and dashboard for audits
- Provide ledger proofs to regulators and customers
