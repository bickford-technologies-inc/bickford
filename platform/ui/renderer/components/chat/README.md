# Bickford Chat Component

This directory contains the Bickford Chat widget for the Bickford Platform UI.

## Features

- Floating chat widget styled for platform consistency
- "Coming soon!" placeholder for future chat functionality
- Easy integration: import and render `<ShowBickfordChat />` anywhere in your UI

## Usage

```tsx
import ShowBickfordChat from "./components/ShowBickfordChat";

function App() {
  return (
    <>
      {/* ...other UI... */}
      <ShowBickfordChat />
    </>
  );
}
```
