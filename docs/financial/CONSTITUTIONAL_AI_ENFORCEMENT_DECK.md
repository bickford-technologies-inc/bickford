## Appendix: Technical Deep Dive

### Canon Rules: Constitutional AI Enforcement Mechanism

**Example: Healthcare AI Deployment**

```typescript
// Constitutional AI Principle: "Do not access patient data outside of immediate clinical need"

// Bickford Canon Rule (Mechanically Enforceable):
const healthcareCanon = {
  scope: "clinical_decision_support",
  authorities: {
    allowed: [
      "read_patient_record(patient_id, context='active_treatment')",
      "write_clinical_note(patient_id, note_type='progress')"
    ],
    prohibited: [
      "read_patient_record(patient_id, context='research')",
      "export_patient_data(destination='external')",
      "access_records(query='all_patients')"
    ]
  },
  audit_triggers: ["data_access", "scope_boundary_test"],
  enforcement: "block_with_audit" // Cannot exceed scope, all attempts logged
};
```

**Result:**
- Constitutional AI principle → mechanically enforceable runtime rule
- Violations are architecturally impossible, not statistically unlikely
- Cryptographic audit trail provides HIPAA compliance proof

### Append-Only Ledger: Cryptographic Audit Trail

**Structure:**
```typescript
interface AuditEntry {
  timestamp: string;           // ISO 8601
  decision_id: string;         // Unique identifier
  canon_rule: string;          // Which rule was enforced
  action_attempted: string;    // What the AI tried to do
  enforcement_result: "allowed" | "blocked";
  cryptographic_hash: string;  // SHA-256 of entry + previous hash
  signature: string;           // Digital signature for non-repudiation
}
```

**Value:**
- Immutable record of every AI decision
- Forensic-grade evidence for security investigations
- Automated compliance report generation (SOC-2, ISO 27001, NIST 800-53)

### OPTR: Optimal Path to Realization

**Decision Authority Optimization:**
1. AI proposes action
2. OPTR validates against Canon rules
3. If allowed: Execute with audit entry
4. If prohibited: Block and log attempted violation
5. If ambiguous: Escalate to human decision authority

**Performance:**
- <50ms latency overhead for governance enforcement
- Parallel execution where Canon rules permit
- Real-time decision timeline visualization for operators
