# Bickford Anthropic Demo App

This app demonstrates Constitutional API Keys and governance enforcement for Anthropic API via Bickford + Bun.

## Features

- Live chat with Anthropic (Claude) via governed API
- Canon enforcement: forbidden actions are blocked
- Tamper-evident ledger for all completions
- Demo UI for acquisition and compliance

## Quick Start

```bash
bun run dev
# or
bun run start
```
